# JavaScript中的设计模式
设计模式往往是软件设计中的最佳实践，是前人对问题解决的经验总结。很多时候你可能会发现在别人的代码会有一些和自己想法不一样的地方，但为什么不知道为什么要这样写，那么本文将会帮助你更好的理解。同时这也是适合新手和老鸟都适合的文章，对于新手而言可以开放自己的思维，对于老鸟而言在遇到问题的时候翻翻，可能也能获得新的思路。

那么现在就开始对JavaScript设计模式的讲解吧。
